## Task 04：实现 Lab 自动校色算法

状态：未开始

### 要求

- 不允许简单 RGB 替换
- 不允许整图套滤镜
- 校色只能作用于目标图蒙版区域
- 标准图也必须使用蒙版区域提取参考颜色
- 目标图的亮度 L 通道要尽量保留
- 主要迁移 Lab 色彩空间中的 a / b 色彩通道
- 支持 colorStrength 参数，范围 0-100
- 支持 shadowProtection
- 支持 highlightProtection
- 支持 maskFeather
- 算法必须写在 src/core/colorTransfer.ts
- UI 组件只负责调用算法和显示结果

### 验收

- 只改变衣服区域颜色
- 不明显破坏纹理细节
- 背景、皮肤、头发等非蒙版区域不变
- 处理结果可以预览
- npm run build 成功
